关于OPML文件

OPML (Outline Processor Markup Language) 是一种用于存储和交换订阅源的标准格式。

支持的格式
  • 标准OPML 1.0/2.0格式
  • 包含Twitter RSS订阅源的文件
  • 支持 xgo.ing API 格式
  • 其他RSS阅读器导出的OPML文件
示例文件

项目中包含以下示例OPML文件：

twitter_ai_feeds.opml
AI领域专家的Twitter RSS订阅源
sample_feeds.opml
基本示例订阅源
注意：导入过程中，已存在的相同URL订阅源将被跳过，不会重复添加。
